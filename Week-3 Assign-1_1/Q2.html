What is CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the visual appearance and formatting of a document written in markup languages like HTML. CSS allows web designers to control the layout, colors, fonts, and other aspects of the presentation of a web page.

Types of CSS

    Inline CSS

    Inline CSS is applied directly to individual HTML elements using the "style" attribute. It is written within the HTML tag itself. Inline CSS has the highest specificity, meaning it will override other CSS rules.

    Internal CSS

    Internal CSS is defined within the head section of an HTML document using the "style" tag. It affects the elements specified within the HTML document.

    External CSS

    External CSS is defined in a separate CSS file and linked to the HTML document using the "link" tag. It allows the same CSS rules to be applied to multiple HTML documents, promoting reusability and ease of maintenance.
